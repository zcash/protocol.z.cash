The Zcash Protocol

This website serves as a router for Zcash protocol identifiers: static strings that refer to a specific part of some Zcash protocol specification or consensus rule, such as TCR:bad-tx-version-group-id. This website acts as a stable URL for these identifiers, and generates links pointing to their canonical specification, as well as search URLs for the zcashd and zebrad codebases.

This website is aware of the following specification locations:

  • The Zcash Protocol Specification
  • Zcash Improvement Proposals (ZIPs)
